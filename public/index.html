                  🎥 Video → looping WebP / MP4
                  
      
                  WebP = small looping cover card for on-chain (native timing). MP4 = high-res, watermarked, constant-frame-rate promo source for Kdenlive/social (editors mishandle WebP's variable timing; add audio in the editor). Keep it short — length drives file size most.
